Learn Microsoft Word 2016

Microsoft Word is a word processing application developed by Microsoft. It is a part of the Microsoft Office suite of productivity software, and it is widely used for creating documents such as letters, resumes, reports, and more. Below are some basics to learn Microsoft Word 2016:

Step 1: Open Microsoft Word 2016 and select blank document

Step 2: In this home tab, you can adjust font, align, sort, borders

Step 3: This font dialog box will be openned by click the left bottom corner arrow of the font section

Step 4:This paragraph dialog box will be openned by click the left bottom corner arrow of the paragraph section

Step 5: Can insert table, pictures, shapes using insert tab

Step 6: Can design the word document using design tab (add water mark, page color, page border etc )

Step 7: Can adjust page layouts like margin orientation, column, size, indet and spacing using layout tab

Step 8: Can mail the document to different receipient
