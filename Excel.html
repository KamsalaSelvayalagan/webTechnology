Learn Microsoft Excel 2016

Microsoft Excel is a spreadsheet program developed by Microsoft. It is a part of the Microsoft Office suite of productivity software, and it is widely used for tasks involving data organization, analysis, and visualization. Below are some resources to help you learn Microsoft Excel 2016:

Step 1: Open Microsoft Excel 2016 and select blank workbook

Step 2: Can create spead sheets

Step 3: Using Page Layout tab, can adjust the page margin, colour, size, orientation etc

Step 4: In formula tab, there are a lot of functions and formulas to do calculations easily

Step 5: Can import data from access, web, text using the options in the Data tab
